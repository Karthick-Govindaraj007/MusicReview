About This Project

This project enhances the Pitchfork Data Lab Project by enabling comprehensive search capabilities for specific artists on a variety of music-related metrics, calculating their average review scores, and identifying their highest and lowest rated projects. Additionally, it provides users with information on the artists' most recent releases or reviews.

Despite the popularity of websites such as Pitchfork as music review platforms, they often lack an accessible view of all scores for every single artist. This project addresses this gap by offering insights into every artist's rating. By incorporating user input to search for these artists, along with utilizing HTML, CSS, JavaScript, and a comprehensive database, the project delivers a functional, user-friendly website. The extension of the original project ensures a more interactive and informative experience for users seeking to explore Pitchfork's extensive review database, using user input to enhance the original design.

The importance of this project lies in its ability to give users access to detailed music review information. For music enthusiasts, researchers, and other professionals in the industry, understanding an artist's reception over time is crucial. By making this information readily available and easy to navigate, the project enhances users' ability to make decisions and discover new artists.
